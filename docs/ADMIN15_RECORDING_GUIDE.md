# 行政 15 句錄影說明

## 目標

這次要補錄 15 句校園行政句，並依照前次整理好的台灣手語詞序，同步建立完整句與句內詞的錄影資料。

- 錄影人數：`3` 人
- 完整句數量：`15` 句
- 每句重複次數：`5` 遍
- 單詞錄影：每句中的每個詞都錄 `5` 遍

## 錄影成員與代號

- `LMX`：梁曼璇
- `XZQ`：徐姿琪
- `XCH`：謝承翰

## 檔名規則

### 完整句

放在：

```text

```

檔名：



```text
data/videos/<stage>/admin15/<member_code>_<name>/words/ADM15_Sxx_<sentence>/Wxx_<word>/
```


格式：

```text
ADM15_S<句號>_W<詞號>_<錄影者代號>_<次數>.mp4
```

## 錄影規則

- 每位組員的 15 句完整句都要錄滿 `5` 遍
- 每位組員的每個句內詞都要錄滿 `5` 遍
- 句內詞順沿用前次 `TSL_admin_15_output` 的台灣手語排序，不用回到華語原順序
- 上半身完整入鏡
- 雙手不要出框
- 鏡頭固定、光線穩定、背景盡量單純
- 每支影片開頭與結尾保留 `0.5 ~ 1` 秒空白
- 不要邊錄邊說話

## 數量估算

- 完整句：`15 句 x 5 遍 = 75` 支 / 人
- 句內詞總數：`69` 詞
- 單詞影片：`69 詞 x 5 遍 = 345` 支 / 人
- 每人總量：`75 + 345 = 420` 支
- 三人總量：`420 x 3 = 1260` 支
